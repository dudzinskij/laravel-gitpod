# Laravel Gitpod

## Getting started

0. In your browser navigate to: https://gitpod.io/#https://gitpod.io/#https://github.com/mchljams/laravel-gitpod
1. Open the terminal
2. Change directory to: "laravel"
    ```
    cd laravel
    ```
3. Run the following Commands:
    ```
    mv .env.example .env 
    ```
    ```
    composer install
    ```
    ```
    php artisan key:generate
    ```
    ```
    php artisan serve
    ```
